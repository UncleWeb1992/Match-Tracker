# Match Tracker

Приложение для мониторинга спортивных матчей

## Установка

```
git clone https://github.com/UncleWeb1992/Match-Tracker.git
npm install
```

## Запуск приложения

```
npm run dev
```

### Открыть в браузере  http://localhost:5173/

## Сборка

```
npm run build
```
